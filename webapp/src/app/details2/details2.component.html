Zacharin/Christie Tree Reading Series

Description

Running since May 9, 1980, Tree Reading Series (Tree) is one of Canada's longest-running literary events and an essential part of Ottawa's vibrant literary community. Only Harbour front (Toronto) has run longer. This ADP collection describes all Tree Reading Series events organized online since March 2020.
